# Literati

Literati is a book rating and review web app that stores users' collections of books. 

### Technologies:

- Ruby/Ruby on Rails
- JavaScript/jQuery
- Sass
- Bootstrap
Packages:
- jQuery
- Devise
- Paperclip
- SimpleForm


### Features for the Future:

More content!: 
- Order/search by book title
- Book groups
- Forum for book discussion
- More dynamic animations
